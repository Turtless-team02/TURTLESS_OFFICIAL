## 📌 팀 소개 (About Us)
안녕하세요! **SSRC2팀 / FRC 11379 터틀리스(TURTLESS)** 입니다. 🐢  
거북이는 수천 킬로미터를 이동한 뒤에도 자신이 태어난 해변을 기억하고, 다시 그곳으로 돌아와 알을 낳습니다. 

저희는 이 이야기를 보며 저희 팀과 비슷하다고 생각합니다.

지난해 시작점인 해변에서 출발해 우승이라는 바다를 항해했습니다. 그리고 올해, 새로운 팀원들과 함께 다시 시작점인 해변으로 돌아 왔습니다.

처음 로봇을 만들던 그 열정, 서로를 믿으며 기본을 쌓아올리던 마음, 그리고 우승을 만들어 냈던 그 과정으로 돌아온 것입니다. 이제 새롭게 함께 하는 팀원들과 한 팀이 되어, 새끼 거북이들이 수많은 포식자의 위협속에서도 끝까지 바다를 향해 나아가듯, 다른 팀과의 경쟁과 수많은 도전을 두려워하지 않고 다시 한 번 우승이라는 바다를 향해 나아갈 것입니다.

우리는 우승했던 팀이 아니라, 우승을 만들어 낸 팀, TURTLESS입니다.
